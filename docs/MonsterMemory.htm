MonsterMemory

these bits represent the monster's memory

Values

Name Description
MEMORY_CLEAR No memory.
bits_MEMORY_PROVOKED right now only used for houndeyes.
bits_MEMORY_INCOVER monster knows it is in a covered position.
bits_MEMORY_SUSPICIOUS Ally is suspicious of the player, and will move to provoked more easily
bits_MEMORY_PATH_FINISHED Finished monster path (just used by big momma for now)
bits_MEMORY_ON_PATH Moving on a path
bits_MEMORY_MOVE_FAILED Movement has already failed
bits_MEMORY_FLINCHED Has already flinched
bits_MEMORY_KILLED HACKHACK -- remember that I've already called my Killed()
bits_MEMORY_MOVE_RADIUS_ONLY Can only get there within radius
bits_MEMORY_CUSTOM4 Monster-specific memory
bits_MEMORY_CUSTOM3 Monster-specific memory
bits_MEMORY_CUSTOM2 Monster-specific memory
bits_MEMORY_CUSTOM1 Monster-specific memory
