RELATIONSHIP

Relationship between (monster) entities

Values

Name Description
R_AL Ally. Good alternative to R_NO when applicable.
R_NO No relationship. Disregard.
R_DL Dislike. Will attack.
R_HT Hate. Will attack this character instead of any DISLIKE'd characters.
R_NM Nemesis. A monster will ALWAYS attack its nemesis, no matter what.
R_FR Fear. Will attack.
