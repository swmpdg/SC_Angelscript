ItemFlag

Item flags used by ItemInfo::iFlags

Values

Name Description
ITEM_FLAG_SELECTONEMPTY Allow this weapon to be selected if it has no ammo
ITEM_FLAG_NOAUTORELOAD Don't automatically reload this weapon
ITEM_FLAG_NOAUTOSWITCHEMPTY Don't automatically switch away from this weapon when it runs out of ammo
ITEM_FLAG_LIMITINWORLD If there are few entity slots left, this item will not respawn
ITEM_FLAG_EXHAUSTIBLE A player can totally exhaust their ammo supply and lose this weapon
