TempEntityType

Temporary entity types. Used in network messages of type SVC_TEMPENTITY.

Values

Name Description
TE_BEAMPOINTS Beam effect between two points
TE_BEAMENTPOINT Beam effect between point and entity
TE_GUNSHOT Particle effect plus ricochet sound
TE_EXPLOSION Additive sprite, 2 dynamic lights, flickering particles, explosion sound, move vertically 8 pps
TE_TAREXPLOSION Quake1 "tarbaby" explosion with sound
TE_SMOKE Alphablend sprite, move vertically 30 pps
TE_TRACER Tracer effect from point to point
TE_LIGHTNING TE_BEAMPOINTS with simplified parameters
TE_BEAMENTS Beam effect between two entities
TE_SPARKS Spark effect
TE_LAVASPLASH Lava splash effect
TE_TELEPORT Teleport effect
TE_EXPLOSION2 Explosion effect
TE_BSPDECAL Decal from the .BSP file
TE_IMPLOSION Tracers moving toward a point
TE_SPRITETRAIL Line of moving glow sprites with gravity, fadeout, and collisions
TE_SPRITE Additive sprite, plays 1 cycle
TE_BEAMSPRITE A beam with a sprite at the end
TE_BEAMTORUS Screen aligned beam ring, expands to max radius over lifetime
TE_BEAMDISK Disk that expands to max radius over lifetime
TE_BEAMCYLINDER Cylinder that expands to max radius over lifetime
TE_BEAMFOLLOW Create a line of decaying beam segments until entity stops moving
TE_GLOWSPRITE Glowing sprite
TE_BEAMRING Connect a beam ring to two entities
TE_STREAK_SPLASH Oriented shower of tracers
TE_DLIGHT Dynamic light effect
TE_ELIGHT Point entity light, no world effect
TE_TEXTMESSAGE Text message
TE_LINE Draws a line
TE_BOX Draws a box
TE_KILLBEAM Kill all beams attached to entity
TE_LARGEFUNNEL Creates a large funnel
TE_BLOODSTREAM Particle spray
TE_SHOWLINE Line of particles every 5 units, dies in 30 seconds
TE_BLOOD Particle spray
TE_DECAL Decal applied to a brush entity (not the world)
TE_FIZZ Create alpha sprites inside of entity, float upwards
TE_MODEL Create a moving model that bounces and makes a sound when it hits
TE_EXPLODEMODEL Spherical shower of models, picks from sets
TE_BREAKMODEL Box of models or sprites
TE_GUNSHOTDECAL Decal and ricochet sound
TE_SPRITE_SPRAY Spray of alpha sprites
TE_ARMOR_RICOCHET Quick spark sprite, client ricochet sound.
TE_PLAYERDECAL Draw a player decal (spray paint)
TE_BUBBLES Create alpha sprites inside of box, float upwards
TE_BUBBLETRAIL Create alpha sprites along a line, float upwards
TE_BLOODSPRITE Spray of opaque sprite1's that fall, single sprite2 for 1..2 secs (this is a high-priority tent)
TE_WORLDDECAL Decal applied to the world brush
TE_WORLDDECALHIGH Decal (with texture index > 256) applied to world brush
TE_DECALHIGH Same as TE_DECAL, but the texture index was greater than 256
TE_PROJECTILE Makes a projectile (like a nail) (this is a high-priority tent)
TE_SPRAY Throws a shower of sprites or models
TE_PLAYERSPRITES Sprites emit from a player's bounding box (ONLY use for players!)
TE_PARTICLEBURST Very similar to lavasplash.
TE_FIREFIELD Makes a field of fire.
TE_PLAYERATTACHMENT Attaches a TENT to a player (this is a high-priority tent)
TE_KILLPLAYERATTACHMENTS Will expire all TENTS attached to a player.
TE_MULTIGUNSHOT Much more compact shotgun message
TE_USERTRACER Larger message than the standard tracer, but allows some customization.
