CommonEntityFlags

Common entity flags used by various entities.

Values

Name Description
SF_NORESPAWN set this bit on guns and stuff that should never respawn.
SF_DELAYREMOVE A weapon will be removed from the map after 60sec
SF_CREATEDWEAPON this flag indicates that this entity is a weapon created for players,
as opposed to being created by being placed by a mapper
SF_DODAMAGE this flag indicates that a player thrown weapon should deal damage to things it hits
SF_GIVENITEM This item is being given to the player -- can't fall out of map
SF_NODEFAULTAMMO Don't allow default ammo to be set. Prevents players from getting too much ammo from cross map inventory
