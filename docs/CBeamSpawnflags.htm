CBeamSpawnflags

CBeam spawn flags

Values

Name Description
SF_BEAM_STARTON Whether the beam should start on
SF_BEAM_TOGGLE Whether the beam should be togglable by triggering it
SF_BEAM_RANDOM Whether beams are created to a randomly selected entity with the given target name
SF_BEAM_RING Whether this beam is a ring between 2 points
SF_BEAM_SPARKSTART Whether to allow creation of sparks at the beam's start point
SF_BEAM_SPARKEND Whether to allow creation of sparks at the beam's end point
SF_BEAM_DECALS Whether to create decals when the beam does damage to BSP models
SF_BEAM_SHADEIN Whether to fade the beam in at its start point
SF_BEAM_SHADEOUT Whether to fade the beam out at its end point
SF_BEAM_TEMPORARY Whether the beam is a temporary entity
