bits_SOUND

Sound flags for monster schedules.

Values

Name Description
bits_SOUND_NONE
bits_SOUND_COMBAT gunshots, explosions
bits_SOUND_WORLD door opening/closing, glass breaking
bits_SOUND_PLAYER all noises generated by player. walking, shooting, falling, splashing
bits_SOUND_CARCASS dead body
bits_SOUND_MEAT gib or pork chop
bits_SOUND_DANGER pending danger. Grenade that is about to explode, explosive barrel that is damaged, falling crate
bits_SOUND_GARBAGE trash cans, banana peels, old fast food bags.
bits_SOUND_BULLETHIT a bullet just hit at this location
bits_ALL_SOUNDS All sound flags
