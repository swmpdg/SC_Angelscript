MoveFlag

these MoveFlag values are assigned to a WayPoint's TYPE in order to demonstrate the
type of movement the monster should use to get there.

Values

Name Description
bits_MF_TO_TARGETENT local move to targetent.
bits_MF_TO_ENEMY local move to enemy
bits_MF_TO_COVER local move to a hiding place
bits_MF_TO_DETOUR local move to detour point.
bits_MF_TO_PATHCORNER local move to a path corner
bits_MF_TO_NODE local move to a node
bits_MF_TO_LOCATION local move to an arbitrary point
bits_MF_IS_GOAL this waypoint is the goal of the whole move.
bits_MF_DONT_SIMPLIFY Don't let the route code simplify this waypoint
bits_MF_TO_JUMP Jumping...
bits_MF_TO_LADDER Ladder...
bits_MF_TO_ELEVATOR Elevator
