CLASS

Entity classification
Can be set on entities with KeyValue using key 'classify'.

Values

Name Description
CLASS_FORCE_NONE Forces class override to none
CLASS_NONE No classification
CLASS_MACHINE Is machine
CLASS_PLAYER Is player
CLASS_HUMAN_PASSIVE Is passive human
CLASS_HUMAN_MILITARY Is military human
CLASS_ALIEN_MILITARY Is military alien
CLASS_ALIEN_PASSIVE Is passive alien
CLASS_ALIEN_MONSTER Is monster alien
CLASS_ALIEN_PREY Is prey alien
CLASS_ALIEN_PREDATOR Is predator alien
CLASS_INSECT Is insect
CLASS_PLAYER_ALLY is player ally
CLASS_PLAYER_BIOWEAPON Is player bioweapon
CLASS_ALIEN_BIOWEAPON Is alien bioweapon
CLASS_XRACE_PITDRONE Is X-Race pitdrone
CLASS_XRACE_SHOCK Is X-Race shock trooper (also Voltigores)
CLASS_BARNACLE Is Barnacle
